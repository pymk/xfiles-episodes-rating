# The X-Files Episodes Rating

I found [this](https://github.com/zimolzak/xfiles) very old repo with a dataset for individual (not average) IMDB user ratings for X-Files episodes. I cleaned it up a bit and merged it with the episode titles and some other metadata from Wikipedia tables.

Datasets are converted to parquet files. The final output is using the `gt` package.
